## サイト概要
本サイトは、エンジニアおよびライターとしての活動実績をまとめたポートフォリオサイトです。
閲覧者に負担をかけない情報設計を軸に、スキル・経験・制作実績を体系化。
必要な情報へ迷わずアクセスできる構成です。

継続的な改善を前提とし、実績を随時更新しています。
ご相談・ご依頼は、InstagramまたはX（旧Twitter）のDMよりご連絡ください。

## 強み
情報を構造化し、分かりやすく設計することを強みとしています。
技術実装と文章制作の両面から、「意図が正確に伝わるアウトプット」を追求しています。
